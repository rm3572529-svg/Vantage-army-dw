Part 1 — Firebase Setup (Backend)

Firebase aapka database aur login system hai. Yeh Google ka free service hai. Koi server khareedne ki zaroorat nahi.

1
Firebase Project Banao

Browser mein yeh link kholo:

https://console.firebase.google.com
  • Apne Google account se login karo
  • "Add project" button pe click karo
  • Project ka naam likho — jaise vantagedown
  • Google Analytics: OFF kar do (baad mein enable kar sakte hain)
  • "Create project" karo — 30 seconds wait karo

Project ban gaya! Ab aap Firebase console mein hoge.

2
Web App Register Karo — Config Copy Karo
  • Firebase Console mein Project Settings (gear icon ⚙️) pe click karo
  • Neeche scroll karo — "Your apps" section mein </> (Web) icon pe click karo
  • App nickname: vantagedown-web likho
  • "Register app" karo
  • Ab ek code box aayega — usmein firebaseConfig dikhega

Yeh config copy karna hai — kuch aisa dikhega:

// Firebase config — yeh aapka unique config hai const firebaseConfig = { apiKey: "AIzaSyCynjRnayg2lCHCBDrLNhw8zREwswVifMg", authDomain: "vantage-share.firebaseapp.com", databaseURL: "https://vantage-share-default-rtdb.firebaseio.com", projectId: "vantage-share", storageBucket: "vantage-share.firebasestorage.app", messagingSenderId: "670971388059", appId: "1:670971388059:web:c9fa7089671543092a263e" };

Yeh config apne paas save kar lo (screenshot lo ya notepad mein paste karo). Agla step mein kaam aayega.

3
firebase.js File Mein Config Paste Karo

ZIP se nikali files mein firebase.js file hai. Use Notepad (Windows) ya TextEdit (Mac) mein kholo.

File ke top par yeh dikhega:

// REPLACE THESE with your Firebase project credentials const FIREBASE_CONFIG = { apiKey: "YOUR_API_KEY", ← Yahan apna paste karo authDomain: "YOUR_PROJECT_ID.firebaseapp.com", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT_ID.appspot.com", messagingSenderId: "YOUR_MESSAGING_SENDER_ID", appId: "YOUR_APP_ID" };

"YOUR_API_KEY" etc. ko apne Firebase se copy kiye values se replace karo.

Sirf values badalni hain — baaki code mat chhedo. File save karo (Ctrl+S).

4
Authentication Enable Karo (Login System)

Firebase Console mein left sidebar mein "Authentication" pe click karo.

  • "Get started" pe click karo
  • "Sign-in method" tab kholo
  • Google pe click karo → Enable karo → Project support email add karo → Save
  • Email/Password pe click karo → Enable karo → Save

Login system ready! Ab Google se aur email se login hoga.

5
Firestore Database Banao

Left sidebar mein "Firestore Database" pe click karo.

  • "Create database" pe click karo
  • Location: asia-south1 (Mumbai) choose karo — India ke liye fastest
  • Security rules: "Start in test mode" choose karo (baad mein proper rules lagayenge)
  • "Enable" karo

Test mode 30 days ke baad expire hoti hai. Baad mein proper security rules daalni hongi — SETUP-GUIDE.html mein rules diye hain.

Proper Security Rules (30 days baad lagao):

Firestore Console → Rules tab → Yeh paste karo:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{uid} { allow read, write: if request.auth != null && request.auth.uid == uid; allow read, write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin'; } match /links/{linkId} { allow read: if true; allow create: if request.auth != null; allow update, delete: if request.auth != null && (resource.data.uid == request.auth.uid || get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin'); } match /config/{docId} { allow read: if true; allow write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin'; } } }

Part 2 — Website Live Karo (Hosting)

Hosting matlab aapki files internet pe upload hoti hain taki koi bhi access kar sake. Hum Netlify use karenge — bilkul free hai, koi credit card nahi chahiye.

6
Netlify pe Upload Karo (Sabse Aasan Tarika)

Yeh steps follow karo:

  • Browser mein kholo: netlify.com
  • "Sign up" karo — GitHub ya Google se login kar sakte hain
  • Dashboard pe aaoge — wahan ek bada dotted box dikhega
  • Uspe likha hoga: "Drag and drop your site folder here"
  • Apni vantage-download-site folder ko us box mein drag karke drop karo
  • 10-20 seconds mein deploy ho jayega
  • Ek URL milega — jaise: amazing-site-123.netlify.app

Website live ho gayi! Woh URL share karo — koi bhi access kar sakta hai.

Custom domain chahiye (jaise vantagedown.com)? Netlify → Domain Settings → Add custom domain. Domain GoDaddy/Namecheap se khareedna padega (~$10/year).

7
Firebase mein Apna Domain Add Karo (Important!)

Google login kaam karne ke liye Firebase ko batana padega ki aapki website ka domain kya hai.

  • Firebase Console → Authentication → Settings tab
  • "Authorized domains" section mein apna Netlify URL add karo
  • Jaise: amazing-site-123.netlify.app
  • "Add domain" karo

Yeh step miss karne pe Google login kaam nahi karega — "unauthorized domain" error aayega.

Part 3 — Admin Account Banao

8
Pehla Account Banao
  • Apni live website par jao
  • login.html page pe jao
  • "Create Account" tab pe click karo
  • Apna naam, email, password dalke register karo
  • Ya Google se sign in karo
9
Apne Account ko Admin Banao

Register hone ke baad Firestore mein jaake manually role change karna hoga:

  • Firebase Console → Firestore Database
  • Left mein "users" collection dikhega — uspe click karo
  • Andar ek document hoga (aapka UID) — uspe click karo
  • role field dikhegi jisme likha hoga "user"
  • Us field pe click karo → "admin" likh do → Save/Update
📸 Firebase Console → Firestore → users → [your-uid] → role: "admin"

Ab aap /admin.html access kar sakte hain! Wahan sab settings milenge.

Part 4 — Google AdSense (Already Done!)

AdSense Already Add Ho Gaya Hai!

Aapka AdSense ID ca-pub-6431906375099854 saari HTML files mein automatically add kar diya gaya hai!

Yeh code har HTML file ke <head> mein hai:

<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js ?client=ca-pub-6431906375099854" crossorigin="anonymous"></script>

Ad Units kahan dikhenge?

  • Homepage pe — generator box ke neeche ek banner
  • Download page pe — top banner + sidebar square ad

AdSense Auto Ads use kar rahe hain to manually kuch nahi karna. Agar specific ad units dikhane hain to AdSense console mein jaake ad unit code copy karo aur HTML files mein .ad-slot div ke andar paste karo.

Ad slots HTML mein kahan hain? In files mein search karo ad-slot:

<!-- Yeh placeholder hain — apna AdSense unit code yahan daalo --> <div class="ad-slot ad-slot-banner" id="ad-top">Advertisement</div> <!-- Replace with: --> <ins class="adsbygoogle" style="display:block" data-ad-client="ca-pub-6431906375099854" data-ad-slot="YOUR_AD_SLOT_ID" data-ad-format="auto"></ins> <script>(adsbygoogle = window.adsbygoogle || []).push({});</script>

Part 5 — Social Links Setup Karo

10
YouTube, Instagram, WhatsApp Links Add Karo

Website live hone ke baad Admin Panel se social links set kar sakte hain:

  • Website pe /admin.html pe jao
  • Left sidebar mein "Social Tasks" pe click karo
  • Apna YouTube, Instagram, WhatsApp link paste karo
  • "Save Configuration" karo

Ya phir seedha HTML files mein bhi daal sakte ho — index.html mein footer section mein href="#" ko apne real links se replace karo.

Social Unlock system ON/OFF karna hai to Admin Panel → Settings → "Social Unlock System" toggle karo.

Problem Aa Rahi Hai?

Google login kaam nahi kar raha?

Firebase Console → Authentication → Settings → Authorized Domains mein apna Netlify URL add karo.

Admin panel access denied aa raha hai?

Firestore → users → apna UID → role field → "admin" likho. Phir logout karke login karo.

Download link kaam nahi kar raha?

Google Drive file mein "Anyone with the link" permission ON karo. Private files download nahi honge.

Firestore permission denied error?

Firestore Rules mein "Test mode" hai to 30 din ke baad expire hoti hai. Upar diye proper rules paste karo.

AdSense ads nahi dikh rahe?

AdSense approve hone mein 24-48 ghante lagte hain. Agar naya account hai to pehle approve hona padega. Auto Ads enable karo AdSense console mein.
